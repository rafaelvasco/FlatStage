VETRIS - A Tetris Inspired Game
---

Controls:
---

- **A** - Move Left
- **D** - Move Right
- **S** - Move Down
- **J** - Rotate Counter Clock Wise
- **K** - Rotate Clock Wise
- **Space** - Instant Drop
- **Enter** - Hold/Release Block
- **Escape** - Exit Game
